# Minion Helper Usage

Run

```
./rabbit.sh
```

## Rabbit Vars

- Rabbit Host
- Rabbit User (administrator)
- Rabbit Password (administrator)

## Options

- Check Vhost
  - Vhost name
- Create Vhost
  - Vhost name
- Check Exchange
  - Vhost name
  - Exchange name
- Create Exchange
  - Vhost name
  - Exchange name
  - Exchange type (direct|topic|fanout|headers)
  - Exchange durable (true|false)
- Check Queue
  - Vhost name
  - Queue name
- Create Queue
  - Vhost name
  - Queue name
- Check Queue
  - Vhost name
  - Queue name
- Create Queue
  - Vhost name
  - Queue name
  - Queue durable (true|false)
  - Queue auto-delete (true|false)
